# @swt-labs/claude-code-driver

Claude Code backend driver for stop-wasting-tokens.

**v1.0 ships this package as a stub** that throws `Not implemented` on every call. This reserves the npm name and locks in the contract surface for v1.5.

The full implementation lands in **v1.5**: 12-event hook taxonomy, Agent Teams, isolation modes (REQ-V2-02). Track progress at [docs.stopwastingtokens.dev/v1-5-roadmap](https://docs.stopwastingtokens.dev/v1-5-roadmap/index) and the canonical roadmap at [`docs/roadmap/v1.5.md`](../../docs/roadmap/v1.5.md).

Until then, use [`@swt-labs/codex-driver`](../codex-driver/) for live execution.
