# @swt-labs/ollama-driver

Ollama / open-source backend driver for stop-wasting-tokens.

**v1.0 ships this package as a stub** that throws `Not implemented` on every call. This reserves the npm name and locks in the contract surface for v1.5.

The full implementation lands in **v1.5** (REQ-V2-03). Track progress at [docs.stopwastingtokens.dev/v1-5-roadmap](https://docs.stopwastingtokens.dev/v1-5-roadmap/index) and the canonical roadmap at [`docs/roadmap/v1.5.md`](../../docs/roadmap/v1.5.md).

Until then, use [`@swt-labs/codex-driver`](../codex-driver/) for live execution.
