# Dashboard — Cmd-K

> **Status:** stub — expanded at M2.
>
> **Canonical reference:** [`TDD2.md` §12](../../TDD2.md).

`Cmd-K` (a.k.a. the global action palette) is the dashboard's keyboard-driven navigation surface. Open with `⌘K` / `Ctrl-K`; type to filter actions across panels (phase, plan, agent, log search). The action set is generated from the dashboard's routing config; each action carries a one-line description + a default keyboard shortcut.

M2 ships the initial action set; subsequent milestones add actions as new panels arrive.
