## Tasks

### T1 ‚Äî TOML emitter
Minimal hand-rolled emitter (no @iarna/toml dependency yet) covering scalars, arrays, and tables. Sufficient for agent + permission TOML output. Comprehensive parser is out of scope ‚Äî Codex is the consumer, we only need to write valid syntax.

### T2 ‚Äî Agent TOML writer
For each AgentRole, emit `<config>/agents/<role>.toml` with `model`, `model_reasoning_effort`, `sandbox_mode`, `developer_instructions`, `allowed_mcp_servers`. Plus a separate `[agents]` global block in `config.toml` covering `max_threads` and `max_depth`.

### T3 ‚Äî AGENTS.md block writer
Read existing AGENTS.md, replace any `<!-- SWT BEGIN -->`‚Ä¶`<!-- SWT END -->` block with the new content, leave everything else verbatim. Compute the resulting byte length, warn (but don't error) if the SWT block plus surrounding content exceeds 32 KiB.

### T4 ‚Äî hooks.json writer
Emit a JSON file with six top-level keys (one per event). Each key is an array of hook entries. Validates shape with a Zod schema before writing.

### T5 ‚Äî Skill installer
Recursively copy a source skill directory into `<config>/skills/<name>/`. Remove on uninstall. Idempotent.

### T6 ‚Äî Custom prompt installer
Copy individual prompt files into `<config>/prompts/`. Remove on uninstall.

### T7 ‚Äî Permission profile writer
Append `[permissions.<name>]` tables to `config.toml` (or write to a separate `permissions.toml` companion file). One profile per AgentRole+sandbox combination.

### T8 ‚Äî Spawn wrapper
`spawn(request)` composes flags, invokes `codex exec`, streams stdout, parses NDJSON lines into SwtHandoff envelopes, returns SpawnResult. Uses execa for cross-platform stability.

### T9 ‚Äî Version detection
`detectCodexVersion(): Promise<{ version: string; major: number; minor: number; patch: number } | undefined>` ‚Äî returns undefined when codex is not on PATH.

### T10 ‚Äî Tests
TOML round-trip, AGENTS.md block replace+preserve, hooks.json schema, NDJSON stream parser (good + malformed lines), version regex.

### T11 ‚Äî Commit + summary
`feat(codex-driver): TOML/AGENTS.md/hooks.json emitters + spawn wrapper + skill installer`.
