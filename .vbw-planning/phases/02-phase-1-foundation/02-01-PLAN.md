# Phase 2 Plan: Foundation

Source: `.vbw-planning/research/source-plan.md` § G, Phase 1.

## Workspace topology

Single repo, pnpm workspaces. One published package (`stop-wasting-tokens`) at the workspace root that bundles all internal packages at build time via tsup. Internal packages live under `packages/` and stay `private: true`.

```
stop-wasting-tokens/
├── package.json                    # public entry; bin: ./dist/cli.js
├── pnpm-workspace.yaml
├── tsconfig.base.json
├── tsup.config.ts
├── eslint.config.mjs
├── .prettierrc
├── vitest.config.ts
├── .changeset/config.json
├── .github/workflows/{ci,release,codeql}.yml
├── .github/dependabot.yml
├── packages/
│   ├── core/
│   ├── cli/
│   ├── codex-driver/
│   ├── methodology/
│   ├── artifacts/
│   ├── verification/
│   └── telemetry/
└── ...
```

## Tasks

### T1 — Root scaffolding
- `pnpm-workspace.yaml`
- `package.json` (root, public, name `stop-wasting-tokens`, type module, engines.node ≥ 20.18, bin, files, scripts)
- `tsconfig.base.json` (strict, NodeNext, ES2022, declarations, sourcemaps)
- `.nvmrc` (20)
- `.editorconfig`
- `.prettierrc` (print width 100, single quotes, no semicolons false)

### T2 — Lint
- `eslint.config.mjs` flat config with `@typescript-eslint`, `eslint-plugin-import`, `eslint-config-prettier` last.

### T3 — Test
- `vitest.config.ts` at workspace root with coverage thresholds 60% lines/branches initially.

### T4 — Build
- `tsup.config.ts` at workspace root that emits ESM+CJS dual build of `packages/cli/src/index.ts` to `dist/cli.{mjs,cjs}` plus `.d.ts`.

### T5 — Workspace packages (×7)
For each of `core`, `cli`, `codex-driver`, `methodology`, `artifacts`, `verification`, `telemetry`:
- `packages/<name>/package.json` (private: true, name `@swt-labs/<name>`)
- `packages/<name>/tsconfig.json` extending `../../tsconfig.base.json`
- `packages/<name>/src/index.ts` stub export

### T6 — Changesets
- `.changeset/config.json` (single-package mode; baseBranch main; access public).
- `.changeset/README.md` placeholder.

### T7 — GitHub Actions
- `.github/workflows/ci.yml` — matrix Node 20/22 × Linux/macOS/Windows running install/typecheck/lint/test/build.
- `.github/workflows/release.yml` — Changesets bot for Version Packages PR; on merge to main publishes to npm with provenance.
- `.github/workflows/codeql.yml` — JavaScript/TypeScript analysis on push + weekly cron.

### T8 — Renovate / Dependabot
- `.github/dependabot.yml` covering npm + github-actions.

### T9 — Smoke test (deferred to user)
`pnpm install && pnpm publish --dry-run` locally on user's machine after Phase 2 lands.

### T10 — Phase 2 commit + summary
One commit: `chore(foundation): scaffold pnpm workspace, build, lint, test, CI`. Then SUMMARY.md.
